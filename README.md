# Pisco Sour — Calculadora de receta

Calculadora de ingredientes para preparar un Pisco Sour chileno casero. Ajusta las cantidades según el número de personas, el tamaño del vaso y la graduación del pisco.

**[Ver sitio →](https://hogox.github.io/pisco-sour/)**

---

## Funciones

- **Calculadora de ingredientes** escalada por número de personas (1–20)
- **Tamaño de vaso**: Normal (100 ml) o Catedral (250 ml)
- **Graduación del pisco**: 35° o 40° — ajusta el volumen de pisco manteniendo el mismo nivel de alcohol, compensando con hielo para no alterar la cantidad total del trago
- **Receta paso a paso** para licuadora casera
- **Modo claro / oscuro** con toggle manual y respeto a la preferencia del sistema
- Diseño accesible — todos los textos cumplen contraste WCAG AA

## Receta base (1 persona, vaso 100 ml, pisco 35°)

| Ingrediente   | Cantidad         |
|---------------|------------------|
| Pisco         | 50 ml            |
| Jugo de limón | 20 ml            |
| Azúcar flor   | 1 cucharadita    |
| Hielo         | 1 cubo           |

### Preparación (licuadora)

1. Licúa el jugo de limón con el azúcar por 2 minutos hasta disolver.
2. Agrega el pisco y mezcla 2 minutos más.
3. Agrega el hielo y licúa hasta lograr consistencia frapé.
4. Añade la clara de huevo y licúa exactamente **4 segundos en velocidad baja**.
